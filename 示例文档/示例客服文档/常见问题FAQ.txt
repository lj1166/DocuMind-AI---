智能手环X1 - 常见问题解答（FAQ）

=== 购买与配送 ===

Q1: 在哪里可以购买？
A: 您可以通过以下渠道购买：
- 官方网站：www.smartband.com
- 京东旗舰店
- 天猫旗舰店
- 线下授权店（查询：官网-门店查询）
价格：299元

Q2: 多久能发货？
A: 现货商品48小时内发货，预售商品以页面说明为准。发货后3-7天送达（偏远地区可能延长）。

Q3: 支持货到付款吗？
A: 支持。但建议在线支付，可享受更多优惠。

Q4: 有什么颜色可选？
A: 黑色、白色、粉色、蓝色四种表带颜色可选。主机为黑色统一款。

=== 使用与设置 ===

Q5: 如何充电？
A: 
1. 使用包装内的磁吸充电线
2. 将充电线的磁吸端对准手环背面金属触点
3. 自动吸附，指示灯亮起表示充电中
4. 充满约需2小时，指示灯变绿
5. 建议每周充电2-3次

Q6: 电池能用多久？
A: 
- 典型使用：7天（心率监测、消息通知）
- 重度使用：3-4天（24小时心率、频繁运动、常亮屏幕）
- 省电模式：10天以上

Q7: 如何连接手机？
A:
1. 下载官方APP"SmartBand"
2. 打开手机蓝牙
3. 在APP中点击"添加设备"
4. 选择"SmartBand X1"
5. 手环震动，点击"✓"确认配对
6. 等待数据同步完成

Q8: 可以连接iPhone吗？
A: 可以。支持iOS 11.0及以上版本。但部分功能受iOS限制：
- 无法读取短信内容（仅显示有短信）
- 无法接听电话（只能提醒）
- 部分第三方APP通知可能不支持

Q9: 可以接听电话吗？
A: 不能直接接听。但会震动提醒来电，显示来电号码或联系人姓名。您需要在手机上接听。

Q10: 支持微信通知吗？
A: 支持。在APP中开启"微信通知"权限即可。手环会显示联系人和消息内容（部分内容）。

=== 功能使用 ===

Q11: 如何测心率？
A: 
自动测量：开启后每5分钟自动测量
手动测量：进入心率界面 → 等待30秒 → 显示结果

注意：
- 佩戴紧度适中
- 手环背面贴合皮肤
- 保持静止状态

Q12: 睡眠监测准确吗？
A: 
准确度：约90%
原理：基于心率、运动传感器数据分析
建议：
- 佩戴整晚
- 设置固定作息时间
- 连续使用一周后数据更准

Q13: 可以游泳佩戴吗？
A: 
可以！5ATM防水等级，支持：
- 游泳（浅水）
- 淋浴
- 下雨

不支持：
- 潜水
- 温泉/桑拿（高温）
- 海水（强腐蚀性）

Q14: 运动数据准确吗？
A: 
步数：准确度约95%
距离：基于步长估算，误差±5%
卡路里：参考值，因人而异
心率：误差±5bpm

提升准确度：
- 正确佩戴（不松不紧）
- 校准步长（APP中设置）
- 开启手机GPS（户外运动）

Q15: 如何切换表盘？
A: 
方法1（手环上）：
  长按主屏幕3秒 → 左右滑动选择 → 点击确认

方法2（APP中）：
  表盘商店 → 选择喜欢的 → 同步到手环

支持：
- 内置表盘：20+款
- 在线表盘：100+款（部分收费）
- 自定义表盘：上传照片

=== 故障排查 ===

Q16: 无法开机怎么办？
A: 
原因1：电量耗尽
解决：充电30分钟后长按开机键

原因2：系统死机
解决：长按开机键10秒强制重启

原因3：硬件故障
解决：联系售后维修

Q17: 连接不上手机怎么办？
A: 
步骤1：基础检查
- 确认蓝牙已开启
- 手机和手环距离<2米
- 手环电量>10%

步骤2：重新配对
- APP中删除设备
- 关闭手环蓝牙
- 重启手机和手环
- 重新添加设备

步骤3：仍无法连接
- 检查手机系统版本
- 更新APP到最新版
- 联系客服

Q18: 数据不同步怎么办？
A: 
原因：
- 蓝牙连接断开
- APP后台被清理
- 网络问题

解决：
1. 打开APP → 下拉刷新
2. 设置APP为"允许后台运行"
3. 手动同步：设备页面 → 同步数据

Q19: 心率测不准怎么办？
A: 
检查：
1. 佩戴位置：手腕内侧向上1指位置
2. 佩戴松紧：能插入1根手指
3. 清洁传感器：擦拭背面绿光区域
4. 皮肤状态：避免纹身、疤痕位置

如仍不准：
- 重启手环
- 更新固件
- 联系售后

Q20: 屏幕不亮怎么办？
A:
原因1：省电模式
解决：设置 → 显示 → 关闭省电模式

原因2：抬腕亮屏未开启
解决：APP → 设备设置 → 开启抬腕亮屏

原因3：定时勿扰
解决：检查是否在勿扰时段

原因4：屏幕故障
解决：联系售后

Q21: 表带如何更换？
A:
1. 按住表带连接处的快拆按钮
2. 向外拉出表带
3. 新表带对准卡槽插入
4. 听到"咔"声即安装成功

注意：
- 购买官方或兼容表带（20mm宽度）
- 安装后检查是否牢固

Q22: 可以接收哪些APP通知？
A:
支持（需在APP中开启）：
- 微信
- QQ
- 短信
- 来电
- 邮件
- 钉钉
- 企业微信
- 其他主流APP

不支持：
- 部分小众APP
- 系统通知

Q23: 如何恢复出厂设置？
A:
方法1（手环上）：
  设置 → 系统 → 恢复出厂设置 → 确认

方法2（APP中）：
  设备管理 → 解绑设备（自动恢复出厂）

注意：恢复后所有数据清空，建议先同步数据

Q24: 固件如何升级？
A:
1. 确保手环电量>50%
2. 连接手机
3. APP提示有新版本
4. 点击"立即更新"
5. 保持连接，约5-10分钟
6. 更新完成自动重启

注意：
- 升级过程不要断开连接
- 不要关闭手环
- 升级后部分设置可能重置

Q25: 数据如何导出？
A:
APP中：
个人 → 数据管理 → 导出数据
支持格式：CSV、Excel
包含：运动记录、心率数据、睡眠数据
